# DeepSeek手握700亿杀入AI编程

> 来源: 量子位/证券时报/IT之家 | 热点: DeepSeek完成700亿融资，DeepSeek Code即将上线
> 日期: 2026年5月26日
> 配图: data/images/cover_20260526_deepseek_code.jpg

---

DeepSeek干了三件事，每一件都让同行头皮发麻。

第一件：完成700亿人民币融资，投前估值约450亿美元，国家大基金领投，梁文锋本人掏了200亿跟投。

第二件：V4-Pro模型API永久降价75%，登上全球性价比榜首。

第三件：DeepSeek Code即将上线，由ACM-ICPC世界冠军崔添翼挂帅，直接杀入AI Coding工具赛道。

三件事放在一起看，梁文锋的意图就很清楚了——不是要做一个更好的大模型公司，而是要改写整个AI行业的定价规则和产品形态。

## 700亿弹药怎么花

先说融资。700亿人民币，约合100亿美元，这是中国AI企业迄今最大规模的融资。

投资方阵容也值得细看：国家人工智能产业投资基金洽谈投资约100亿，腾讯、IDG资本、砺思资本已接近确定参投，宁德时代、网易、京东也在对接中。产业资本+国家队+财务资本，三路资金齐聚。

更值得注意的是梁文锋本人掏了200亿。一个创始人拿自有资金跟投，要么是对公司极度自信，要么是担心外部资本稀释控制权。大概率两者兼有。

融资后估值摸到3000亿人民币。这笔钱够干什么？够支撑未来几年的算力基建和技术迭代。在AI行业，算力就是弹药，弹药充足才有资格打持久战。

## 永久降价75%，打的是谁

再说降价。

5月22日，DeepSeek宣布V4-Pro API价格永久降至原价的四分之一。输入缓存命中每百万Tokens只要0.025元，输入缓存未命中3元，输出6元。

第三方机构Artificial Analysis评估后给了个结论：DeepSeek-V4-Pro是全球性价比最高的AI模型。

这个定价意味着什么？对比一下就知道了。亚马逊涨、微软涨、国内主流云厂商也跟着涨，部分API涨幅甚至高达463%。整个行业站在同一个逻辑上：算力贵，所以AI贵。

DeepSeek反其道而行之。别人涨价它降价，别人临时促销它永久降价。

这不是赔本赚吆喝。DeepSeek的技术路线本就以低成本著称——2025年它就是靠成本远低于同行的模型颠覆了硅谷的定价体系。这次降价，本质上是把技术优势转化为价格优势，进一步挤压竞品的生存空间。

## 崔添翼和DeepSeek Code

最值得玩味的是第三件事。

DeepSeek Code即将上线，负责人崔添翼，这个名字在编程圈含金量极高。浙大计算机系毕业，6次ACM-ICPC亚洲区域赛金牌，毕业后在Jane Street香港和纽约做了9年交易系统，后来联合创立TSY Capital做量化交易。

注意这个背景：量化交易老兵，不是传统的大模型研究员。

这说明DeepSeek Code不是简单地把V4模型包装成一个编程助手，而是要从工程实践层面重新定义AI编程工具。崔添翼在Jane Street写过的交易系统，对延迟、可靠性、工程规范的要求，远超一般的互联网应用。

DeepSeek从底层模型走向垂直产品，这个战略转向意义重大。做模型是"卖铲子"，做产品是"自己挖金矿"。当你的模型足够好、足够便宜，为什么不直接做终端产品？

## 一盘更大的棋

把三件事串起来看：

700亿融资解决的是"能不能打持久战"的问题，永久降价解决的是"能不能抢到更多用户"的问题，DeepSeek Code解决的是"能不能从模型公司变成产品公司"的问题。

梁文锋自己说过：技术边界才是第一要务，不急于变现。

但你看他做的每件事，都是在为变现铺路。只不过他的变现方式不是按Token收费，而是用低价模型获取海量用户，再用垂直产品（比如Code）实现商业闭环。

这套打法，有点像当年的亚马逊：先亏本建基础设施（AWS），等生态成型了，利润自然来。

DeepSeek能不能复制这个路径，现在不好说。但有一点可以确定：AI行业的价格战，才刚刚开始。
